# Тестирование фейк-банка

## Описание
Проект содержит ручные и автоматизированные тесты для фейкового банковского сервиса.

## Найденные дефекты
1. Перевод 0 ₽ возможен
2. Пустое поле суммы → перевод 1000 ₽
3. Можно ввести 17 цифр карты
4. Отрицательная сумма перевода принимается
5. Комиссия 99 ₽ = 0 (должна быть 9)

## Автотесты
5 Selenium-тестов на Python + pytest

## CI
GitHub Actions — сборка красная (все тесты падают)
