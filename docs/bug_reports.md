# Баг-репорты

## Баг-репорт №1
- **Issue:** https://github.com/k1onnu/fake-bank-testing_netology/issues/1

| Поле | Значение |
|------|----------|
| **Заголовок** | Перевод 0 ₽ возможен (включая пустое поле) |
| **Серьёзность** | Major |
| **Приоритет** | High |
| **Шаги воспроизведения** | 1. Открыть банк<br>2. Нажать на рублёвый счёт<br>3. Ввести 16-значную карту<br>4. Ввести сумму 0 (или оставить поле пустым)<br>5. Нажать "Перевести" |
| **Ожидаемый результат** | Ошибка "Сумма должна быть больше 0" |
| **Фактический результат** | Перевод 0 ₽ принят банком |
| **Окружение** | Chrome, localhost:8000 |
