# Ручное тестирование фейк-банка

## Тест-кейс №1 (Пройден)
**Название:** Успешный перевод на 16-значную карту  
**Предусловие:** Баланс 30 000 ₽, резерв 20 001 ₽  
**Шаги:**
1. Нажать на рублёвый счёт
2. Ввести номер карты: 1111222233334444 (16 цифр)
3. Ввести сумму: 1000
4. Нажать "Перевести"  

**Ожидаемый результат:** Уведомление "Банк принял запрос на перевод"   
**Фактический результат:** Уведомление получено  [+]
